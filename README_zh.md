# Vue3 CRUD [English](https://github.com/51fe/vue3-crud/blob/master/README.md)
> 使用Vue3 + Vite + TypeScript + element-plus + axios + json-server开发的简单CRUD Web应用。麻雀虽小，五脏俱全。

## 特性
1. 支持CRUD
2. 支持基本的搜索、分页、排序
3. 中国市区级联、自定义组件/数据访问/表单验证
4. 响应式布局
5. 单元测试

## 项目设置
```
npm install
```

### 编译前运行json serve
```
npm run mock
```

### 开发编译热替换
```
npm run dev
```

### 单元测试
```
npm run test:unit
```
